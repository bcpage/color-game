Paradox: Russell's Barber

"The barber shaves all and only those men in town who do not shave themselves."
Who shaves the barber?
Case A: The barber shaves himself. → The barber shaves only those who do NOT shave themselves. → Therefore the barber does not shave himself. Contradiction. Case B: The barber does not shave himself. → The barber shaves ALL those who do not shave themselves. → Therefore the barber shaves himself. Contradiction. Both cases produce contradiction. The barber as described cannot exist.

Formulated by Bertrand Russell in 1901. The barber paradox is a simplified version of Russell's paradox, which showed that naive set theory — the foundation of mathematics as it was understood at the time — is logically inconsistent.

The naive version: let R be the set of all sets that do not contain themselves. Does R contain itself? If yes, by definition it doesn't. If no, by definition it does. Contradiction either way. The set R cannot exist.

This broke Georg Cantor's paradise. Mathematicians had believed that any property could define a set. Russell's paradox proved that some properties define sets that cannot exist.

The resolution — developed by Russell himself and by Ernst Zermelo and Abraham Fraenkel — was to restrict which properties are allowed to define sets. Modern mathematics operates within these restrictions. The paradox does not recur. But it is always one definition away.

The barber cannot exist. The sentence that describes him is grammatically valid and semantically empty.
