# [00062] — Paradox: Russell's Barber

**Status:** Built
**Last updated:** 2026-05-23

## What it is
Russell's barber paradox. The premise ("shaves all and only those who do not shave
themselves"), the question ("who shaves the barber?"), a two-case logic block showing
both cases lead to contradiction, and an account covering Russell's paradox proper,
naive set theory, and the ZFC resolution. Closing line: "The barber cannot exist."

## How it works
Static HTML. The logic block uses monospace spans with red coloring on the contradiction
lines. No interaction. No server calls.
